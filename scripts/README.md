# ¿Cómo usarlos?

* 1. Dale permisos de ejecución:
     
     ```bash
     chmod +x scripts/*.sh
     ```

* 2. Ejecuta cualquier script con:
     
     ```bash
     ./scripts/activar-rsyslog.sh
     ./scripts/enviar-log-scp.sh
     ./scripts/enviar-log-netcat.sh
     ./scripts/escuchar-netcat.sh
    ```
